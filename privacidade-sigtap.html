Aplicativo Android

Política de Privacidade

SIGTAP Procedimentos SUS  ·  Última atualização: 31 de maio de 2025

Resumo: O SIGTAP Procedimentos SUS é um aplicativo de consulta offline. Ele não coleta, armazena nem compartilha dados pessoais dos usuários.

1. Sobre o Aplicativo

O SIGTAP Procedimentos SUS é um aplicativo de referência que permite consultar a Tabela de Procedimentos, Medicamentos e OPM do Sistema Único de Saúde (SUS), conforme disponibilizado pelo Ministério da Saúde. O aplicativo funciona de forma offline após a instalação.

2. Dados Coletados

Este aplicativo não coleta nenhum dado pessoal. Não são solicitados nome, e-mail, localização, contatos, histórico de navegação ou qualquer outra informação identificável.

O sistema operacional Android pode gerar automaticamente relatórios técnicos de falhas (crash reports) que são enviados ao Google Play Console. Esses relatórios contêm dados técnicos do dispositivo (modelo, versão do Android, stack trace) e não incluem informações pessoais identificáveis. Sua coleta é gerida pela política de privacidade do Google.

3. Dados de Uso

Não utilizamos ferramentas de análise de uso (analytics), publicidade ou rastreamento. Nenhum SDK de terceiros para coleta de dados está integrado ao aplicativo.

4. Permissões Solicitadas

O aplicativo pode solicitar as seguintes permissões mínimas:

  • Armazenamento – para salvar a base de dados local de procedimentos, quando aplicável.

Nenhuma permissão de localização, câmera, microfone, contatos ou acesso à internet é solicitada para a funcionalidade principal.

5. Crianças e Adolescentes

Este aplicativo é destinado a profissionais de saúde e estudantes maiores de 18 anos. Não coletamos intencionalmente dados de menores de idade.

6. Lei Geral de Proteção de Dados (LGPD)

Como o aplicativo não coleta dados pessoais, as obrigações previstas na Lei nº 13.709/2018 (LGPD) relativas a tratamento, armazenamento e compartilhamento de dados não se aplicam a este aplicativo de forma direta. Caso isso mude em versões futuras, esta política será atualizada.

7. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que você a revise de tempos em tempos. A data de "última atualização" no topo desta página indica quando a versão mais recente entrou em vigor.
